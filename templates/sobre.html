Sobre o Projeto PIM III

Documentação resumida para apresentação

Objetivo

Desenvolver um sistema web integrado para um negócio fictício, com qualidade, dados, usabilidade e comunicação organizacional.

Tecnologias

Python Flask, SQLite, Bootstrap 5, Chart.js, HTML, CSS e JavaScript.

Funcionalidades

Login seguro, permissões, clientes, produtos, vendas com carrinho, relatórios, estoque baixo, histórico, perfil, modo claro/escuro e dashboard com gráficos.

Acessibilidade

Interface responsiva, contraste visual, ícones, botão VLibras visual e navegação simples.
