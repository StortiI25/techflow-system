Sobre o Projeto PIM III

Resumo técnico do EstoqueFlow.

Objetivo

O EstoqueFlow é um sistema web para gerenciamento de estoque, criado para pequenas empresas controlarem produtos, entradas, saídas, clientes e relatórios.

Tecnologias

Python Flask, SQLite, Bootstrap 5, Chart.js, JavaScript, HTML, CSS e VLibras.

Diferenciais

Auditoria, bloqueio de estoque negativo, estoque crítico, movimentações tipificadas, busca por código de barras, dashboard, exportação CSV/PDF e personalização.
